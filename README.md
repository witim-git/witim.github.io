# 个人主页 - witim.github.io

这是我的个人GitHub Pages主页，展示个人信息、技能、项目和工作经验。

## 📋 主页内容

- **个人简介** - 关于我和职业背景
- **技能展示** - 技术栈和技能等级
- **工作经验** - 职业发展历程
- **代表作品** - 项目和开源贡献

## 🚀 快速开始

1. 访问主页：https://witim-git.github.io
2. 查看 `index.html` 了解页面结构
3. 根据需要自定义内容

## 📝 自定义指南

### 修改个人信息
编辑 `index.html` 中的以下部分：
- 头像和名字（第95行）
- 职位/简介（第96行）
- 联系方式链接（第97-99行）

### 修改技能
在 `index.html` 的技能部分修改：
- 技能名称和进度条宽度

### 更新项目
在项目经验部分添加或修改项目信息
